# Online Grocery Store

This project is an online grocery website where users can browse products, add items to the cart, and view the total bill.

## Features
- List of grocery items
- Add to cart option
- Calculate total price
- Simple and clean user interface
## How to Use
1. Open the home page.
2. Select your grocery items.
3. Add items to the cart.
4. View the final bill.
